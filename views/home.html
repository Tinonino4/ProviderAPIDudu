Bienvenidos a la API de eventos de fútbol de MBIT!!

Esta aplicación está configurada para enviar un evento cada segundo que corresponde a un evento de cada minuto de un partido.

Es decir, si pulsamos el botón se enviarán un total de 90 mensajes a un Azure Storage Container con un intervalo entre mensajes de 1 segundo. Cada mensaje simula un evento en un minuto del partido. Por ejemplo, un mensaje sería: Minuto 10, Accion 'Ataque peligroso'

El objetivo de esta aplicación es simular a una API que nos manda mensajes, y así en lugar de tener que subirlos nosotros a Azure, directamente conectamos nuestra aplicación al contenedor.
